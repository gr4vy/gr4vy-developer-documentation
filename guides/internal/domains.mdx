---
title: Merchant domains
---

By default, every cluster will be available under a Gr4vy subdomain based on the
merchant's `short_name`.

The merchant's `short_name` can be queried by calling the
[`GET /merchants/main`][main_merchant_api] API.

[main_merchant_api]: /internal/reference#operation/get-current-merchant

```json title="GET /merchants/main" {3}
{
  "type": "merchant",
  "short_name": "acme",
  "display_name": "The Acme Corporation"
}
```

With this `short_name` it is possible to determine a set of sub domains used by
a merchant cluster.

|              | URL                                   | Description                                 |
| ------------ | :------------------------------------ | :------------------------------------------ |
| `short_name` | `acme`                                | The short name for a merchant               |
| Base domain  | `https://acme.app.gr4vy.com`          | Redirects to the cluster admin panel        |
| Admin        | `https://admin.acme.app.gr4vy.com`    | The cluster admin panel                     |
| API          | `https://api.acme.app.gr4vy.com`      | The cluster API                             |
| CDN          | `https://cdn.acme.app.gr4vy.com`      | The CDN for the cluster, used to host Embed |
| Redirect     | `https://redirect.acme.app.gr4vy.com` | URL used for browser-based redirects        |
