---
title: Merchant domains
---

By default, every cluster will be available under a Gr4vy subdomain based on the
merchant's `short_name`.

The merchant's `short_name` can be queried by calling the
[`GET /merchants/main`][main_merchant_api] API.

[main_merchant_api]: /reference/internal#operation/get-current-merchant

```json title="GET /merchants/main" {3}
{
  "type": "merchant",
  "short_name": "acme",
  "display_name": "The Acme Corporation"
}
```

With this `short_name` of `acme` it is possible to determine a set of sub
domains used by a merchant cluster as follows.

| URL                                   | Description                                 |
| :------------------------------------ | :------------------------------------------ |
| `https://acme.app.gr4vy.com`          | Redirects to the cluster admin panel        |
| `https://admin.acme.app.gr4vy.com`    | The cluster admin panel                     |
| `https://api.acme.app.gr4vy.com`      | The cluster API                             |
| `https://cdn.acme.app.gr4vy.com`      | The CDN for the cluster, used to host Embed |
| `https://redirect.acme.app.gr4vy.com` | URL used for browser-based redirects        |
