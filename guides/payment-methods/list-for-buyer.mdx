import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

To list all the vaulted payment methods for a buyer, call the
[`GET /buyers/payment-methods`][api] API with the `buyer_id` and/or
`buyer_external_identifier` as a query parameter.

<Tabs
  groupId="language"
  defaultValue="curl"
  values={[
    { label: 'cURL', value: 'curl', },
    { label: 'Node', value: 'node', },
  ]}
>
  <TabItem value="curl">

```bash
curl -i -X GET "https://api.{gid}.gr4vy.app/buyers/payment-methods?buyer_id=fe26475d-ec3e-4884-9553-f7356683f7f9" \
    -H "Authorization: Bearer [JWT_TOKEN]"
```

  </TabItem>
  <TabItem value="node">

```js
const fetch = require("node-fetch");

fetch(
  "https://api.{gid}.gr4vy.app/buyers/payment-methods?buyer_id=fe26475d-ec3e-4884-9553-f7356683f7f9",
  {
    method: "GET",
    headers: {
      Authorization: "Bearer [JWT_TOKEN]",
    },
  }
);
```

  </TabItem>
</Tabs>

The response is a [list of all the previously stored payment methods][api] for
that buyer in a mini format.

```json title="GET /buyers/payment-methods?buyer_id=fe26475d-ec3e-4884-9553-f7356683f7f9"
{
  "items": [
    {
      "type": "payment-method",
      "id": "77a76f7e-d2de-4bbc-ada9-d6a0015e6bd5",
      "method": "card",
      "details": {
        "suffix": "1111",
        "scheme": "visa",
        "expiration_date": "11/25"
      }
    }
  ],
  "limit": 1,
  "next_cursor": "ZXhhbXBsZTE",
  "previous_cursor": null
}
```
