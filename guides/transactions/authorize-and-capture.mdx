By default a transaction is only authorized and not automatically captured. A
transaction can be captured either directly when creating a transaction, or as a
separate API call.

## Simultaneous authorize and capture

It is possible to perform a capture directly when calling
[`POST /transactions`](/reference#operation/authorize-new-transaction) by
setting the `capture` field in the request body to `true`.

<Tabs
  groupId="language"
  defaultValue="curl"
  values={[
    { label: 'cURL', value: 'curl', },
    { label: 'Node', value: 'node', },
  ]}
>
  <TabItem value="curl">

```bash {7}
curl -i -X POST "https://api.{subdomain}.gr4vy.com/transactions" \
    -H "Authorization: Bearer [JWT_TOKEN]" \
    -H "Content-Type: application/json" \
    -d '{
          "amount": 1299,
          "currency": "USD",
          "capture": true
          "payment_method": {
            "method": "card",
            "number": "4111111111111111",
            "expiration_date": "11/25",
            "security_code": "123"
          }
        }'
```

  </TabItem>
  <TabItem value="node">

```js
var fetch = require("node-fetch");

fetch("https://api.{subdomain}.gr4vy.com/transactions", {
  method: "POST",
  headers: {
    Authorization: "Bearer [JWT_TOKEN]",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    amount: 1299,
    currency: "USD",
    capture: true,
    payment_method: {
      method: "card",
      number: "4111111111111111",
      expiration_date: "11/21",
      security_code: "123",
    },
  }),
});
```

</TabItem>
</Tabs>

## Capture after authorization

Alternatively, it is possible to perform a capture on a previously authorized
transaction by calling the
[`POST /transactions/{transaction_id}/capture`](/reference#operation/capture-transaction)
API

<Tabs
  groupId="language"
  defaultValue="curl"
  values={[
    { label: 'cURL', value: 'curl', },
    { label: 'Node', value: 'node', },
  ]}
>
  <TabItem value="curl">

```bash {7}
curl -i -X POST "https://api.{subdomain}.gr4vy.com/transactions/fe26475d-ec3e-4884-9553-f7356683f7f9/capture \
    -H "Authorization: Bearer [JWT_TOKEN]"
```

  </TabItem>
  <TabItem value="node">

```js
var fetch = require("node-fetch");

fetch(
  "https://api.{subdomain}.gr4vy.com/transactions/fe26475d-ec3e-4884-9553-f7356683f7f9/capture",
  {
    method: "POST",
    headers: {
      Authorization: "Bearer [JWT_TOKEN]",
    },
  }
);
```

</TabItem>
</Tabs>
